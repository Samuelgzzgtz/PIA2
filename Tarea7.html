Haskell

Haskell es un lenguaje de programación funcional, es decir, se basa en el cálculo lambda y en la evaluación de funciones matemáticas. Se caracteriza por su fuerte tipado, su inferencia de tipos y su inmutabilidad de datos.

Símbolos

Los símbolos utilizados en Haskell son:

  • Letras y dígitos: a-z, A-Z, 0-9
  • Caracteres especiales: +, -, *, /, %, ^, &, |, ~, !, ?, ., :, =, <, >, @, #, $, \, _, ' y ".
  • Caracteres especiales adicionales utilizados para funciones: ->, <-, ::, =

Sintaxis

La sintaxis de Haskell se basa en expresiones y declaraciones. Una expresión puede ser un valor, una variable, una función o una aplicación de funciones. Una declaración puede ser una asignación, una definición de función o un condicional.

Esqueleto de programas

Un programa básico en Haskell tiene la siguiente estructura:

    
        -- Módulo principal
        module Main where

        -- Importar librerías
        import Data.List

        -- Funciones
        main :: IO ()
        main = do
            putStrLn "Hola, mundo!"

Aplicación para descargar y ejecutar

Existen varias aplicaciones para descargar y ejecutar programas en Haskell. Algunas de las más populares son:

  • GHC (Glasgow Haskell Compiler): compilador de Haskell para sistemas operativos Windows, macOS y Linux.
  • Haskell Platform: plataforma completa que incluye el compilador GHC y varias librerías estándar.
  • OnlineGHC: sitio web que permite compilar y ejecutar programas Haskell en línea.

Tipo de paradigma

Haskell es un lenguaje de programación funcional, lo que significa que se basa en la evaluación de funciones matemáticas. Es un lenguaje de programación de alto nivel que permite la programación declarativa, lo que significa que se enfoca en qué se debe hacer en lugar de cómo se debe hacer.

Ejemplo de programa en Haskell

Aquí tienes un ejemplo de un programa en Haskell que calcula el factorial de un número:


	-- Importamos el módulo Prelude que contiene funciones básicas como (+), (-), (*), etc.
	import Prelude

	-- Definimos una función que toma un número entero y devuelve su factorial
	factorial :: Integer -> Integer
	factorial 0 = 1 -- Caso base
	factorial n = n * factorial (n - 1) -- Caso recursivo

	-- Función principal que se ejecuta al correr el programa
	main :: IO ()
	main = do
		putStrLn "Calculadora de factoriales"
		putStrLn "Ingresa un número:"
		input <- getLine
		let n = read input :: Integer
		let result = factorial n
		putStrLn ("El factorial de " ++ input ++ " es " ++ show result)

En este programa, se define una función factorial que calcula el factorial de un número entero utilizando recursión. Luego, en la función main, se pide al usuario que ingrese un número y se utiliza la función factorial para calcular su factorial, el cual es mostrado en la salida estándar. El esqueleto básico del programa en Haskell incluye la definición de funciones, así como la función main que se ejecuta al correr el programa. También se pueden incluir importaciones de módulos externos y comentarios.
