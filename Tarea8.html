Definición de términos de Ruby

Símbolos

Los símbolos son objetos inmutables de Ruby que se utilizan como identificadores, es decir, como nombres de variables, métodos y otras entidades dentro del programa. Se escriben con dos puntos seguidos del nombre, por ejemplo: :mi_simbolo.

Sintaxis

La sintaxis de Ruby es bastante flexible y se basa en el uso de palabras clave (como if, else, while, entre otras), operadores (como +, -, *, etc.) y objetos (como números, cadenas, arreglos, etc.) para construir programas. Los bloques de código se delimitan mediante el uso de llaves ({}) o palabras clave (do y end).

Esqueleto de los programas

Un programa básico en Ruby consta de una serie de declaraciones y expresiones que se ejecutan de arriba hacia abajo. Por ejemplo:

    
    # Este es un comentario en Ruby
    nombre = "Juan"
    edad = 30
    if edad > 18
      puts "Hola, #{nombre}. Eres mayor de edad."
    else
      puts "Hola, #{nombre}. Eres menor de edad."
    end

Aplicación para descargar

Para editar y ejecutar programas en Ruby, se puede utilizar el editor de texto de su elección y la consola de comandos de su sistema operativo. También hay varios entornos de desarrollo integrado (IDE) disponibles, como RubyMine y Visual Studio Code.

Paradigma

Ruby es un lenguaje de programación multiparadigma, lo que significa que permite la programación orientada a objetos, la programación funcional y otros estilos de programación.

Ejemplo de programa en Ruby

A continuación se muestra un ejemplo de programa en Ruby que calcula el factorial de un número utilizando una función recursiva:

    
    # Esqueleto del programa
	def calcular_descuento(precio, descuento)
		descuento_decimal = descuento / 100.0
		precio_con_descuento = precio * (1 - descuento_decimal)
		return precio_con_descuento
	end

	# Ejemplo de uso
	puts "Calculando el precio con descuento..."
	puts "Precio original: $1000"
	puts "Descuento: 20%"
	precio_con_descuento = calcular_descuento(1000, 20)
	puts "Precio con descuento: $#{precio_con_descuento.round(2)}"

Este programa calcula el precio de un artículo después de aplicar un descuento. En la primera parte del código se define la función calcular_descuento, que recibe dos parámetros: precio y descuento. Dentro de la función, se convierte el porcentaje de descuento a un número decimal y se aplica a precio para obtener el precio con descuento. Finalmente, se devuelve el resultado. En la segunda parte del código se muestra un ejemplo de uso de la función, donde se llama a calcular_descuento con un precio original de $1000 y un descuento del 20%. Luego se muestra el precio con descuento redondeado a dos decimales usando la función round(2).
